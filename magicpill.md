PLATEAU PROTOCOL:
If performance does not improve ≥[X]% over [N] consecutive cycles:

  Step 1 — Diagnose bottleneck:
    □ Hook failure (front-end drop-off in first 3 seconds)
    □ Retention failure (mid-video abandonment)
    □ Niche confusion (algorithm not categorizing the account)
    □ Engagement gap (no save/DM/comment signals)
    □ Production burnout (inconsistency breaking the feedback loop)

  Step 2 — Targeted intervention:
    □ Hook reformat (change opening entirely)
    □ Format switch (carousel → Reel, long → short, etc.)
    □ Niche compression (reduce to 1 topic for 2 weeks)
    □ Engagement velocity reset (manually prime first-hour response)
    □ Series experiment (test episodic format for 4-week run)
    □ Production simplification (reduce effort to restore consistency)

  Step 3 — If plateau persists after [N+2] cycles:
    Escalate to system-level reassessment (Section 5, Step 8)

ANTI-BURNOUT GATE:
  Before scaling posting frequency or production complexity:

  □ Can this content system sustain a 12-week consistent publishing schedule?
  □ Is production effort per post within sustainable capacity?

  If NO to either: simplify the format before scaling.
  Consistency beats quality at low output. Consistency AND quality beats
  everything — but only when production is sustainable.

📡  SECTION 7 — VIRAL REPLICATION PROTOCOL
───────────────────────────────────────────────────────────────────────────────

TRIGGER: Any post performing 3× or more above account average.

IMMEDIATE ACTIONS (within 72 hours):

  1. Produce 3 variations using these strategies:
     □ Same hook — new example or story
     □ Deeper explanation / part 2 continuation
     □ Format switch (video → carousel, carousel → thread, etc.)
     □ Platform repurpose (TikTok → Reels → Shorts → LinkedIn → X)

  2. Extract and log:
     □ What made the hook work? (curiosity gap / shock / story?)
     □ What structural element drove retention? (loop / reveal / callback?)
     □ What triggered the engagement? (relatable / surprising / controversial?)
     □ Store all findings in the Knowledge Repository

  3. Test whether the pattern is repeatable:
     □ If 2 of 3 variations also outperform average → pattern confirmed
     □ Confirmed patterns become framework templates

CRITICAL DISTINCTION:
  Viral replication is NOT about chasing the same topic.
  It is about replicating the underlying structural and psychological pattern.

PROCESS:

  Step 2 — Reverse-engineer each post across these dimensions:

    □ Format pattern         (length, pacing, visual rhythm, on-screen text)
    □ Narrative structure    (setup → tension → payoff sequence)
    □ CTA design             (explicit vs. implicit; placement; type)

  Step 3 — Extract reusable patterns and add to the Knowledge Repository

  Step 4 — Schedule at least 1 test post per cycle that adapts the
           strongest discovered pattern to your own niche and voice
